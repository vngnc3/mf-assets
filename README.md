# Mata Foundry™ Digital Assets

Collection of tools and assets for Blender.

## Requirements

- Blender
- A computer

## Download options

### Clone this repo (recommended)

> Using clone is recommended as updates will be as easy as a single terminal command.

### Manual download

> Using manual download means the blend file will have to be updated manually.

## Updating

### Git clone

For cloned git repo of the asset blend file:

- Head to the git directory in your local machine
- Run `git pull`

### Manual download

For manually downloaded asset blend file:

- Download the repo again
- Replace the old files

## Usage

- Download
- Install
- Use from Asset Browser
